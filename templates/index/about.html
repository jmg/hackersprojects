About

Hackers Projects is a github projects aggregator where the developers are able to share, vote, comment and contribute open source projects generated by the community itself. It's a kind of hackersnews-like site but for open source projects hosted on github.

It is and it will always be completely open source. You can contribute with bug fixing, new features, a better design (I'm a developer and my lack of design / css abilities are certainly reflected on the bootstrapy aspect this site has right now). The gihub repository is the following https://github.com/jmg/hackersprojects. Feel free to send me a pull request and let's make this project grow as a community of open source projects and developers!

The idea is to fund this project via donations. Read more on the donations page.
